POST to https://fakestoreapi.com/products

Body

{
    "title": 12345,
    "price": "not-a-number",
    "description": true
}

Headers

Accept: application/json, application/javascript, text/javascript, text/json
Content-Type: application/json

Curl

curl -v -X POST 'https://fakestoreapi.com/products' -H 'Accept: application/json, application/javascript, text/javascript, text/json' -H 'Content-Type: application/json' -d '{ "title": 12345, "price": "not-a-number", "description": true }'
